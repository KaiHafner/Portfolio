Reptile Wipeout

Unreal Engine Blueprints Game Design Level Design

Project Overview

Reptile Wipeout is a fast-paced obstacle course game where players race against time to navigate through treacherous paths filled with deadly traps and environmental hazards. The game challenges players' reflexes and precision as they attempt to reach the finish line.

As the lead developer on this project, I was responsible for the core gameplay mechanics, level design, and visual effects. The game features dynamic obstacles that require quick thinking and precise timing to overcome.

The development process involved creating a responsive control system that provides tight feedback to player inputs, essential for a game that relies heavily on precision movement. I implemented a checkpoint system that balances challenge with accessibility, allowing players to progress through difficult sections without excessive frustration.

My Contributions

  • Designed and implemented core gameplay mechanics including movement, jumping, and obstacle interaction
  • Created multiple challenging levels with increasing difficulty curves
  • Developed dynamic trap systems with varying patterns and timings
  • Implemented visual and audio feedback systems to enhance player experience
  • Optimized performance for smooth gameplay across different hardware specifications
  • Integrated particle effects and environmental animations to bring levels to life
  • Designed and implemented the user interface including menus and HUD elements

Pirate Survivors

Unreal Engine Blueprints AI Programming Procedural Generation

Project Overview

Pirate Survivors is a survival game set in the treacherous waters of the Caribbean where players take on the role of a pirate captain searching for Davy Jones' legendary loot. The game combines exploration, combat, and resource management as players navigate dangerous seas filled with supernatural threats.

My role in this project focused on enemy AI behavior, weapon systems, and the procedural generation of ocean environments. I worked extensively on creating engaging combat encounters that challenge players to adapt their strategies based on enemy types and environmental conditions.

The game features a dynamic weather system that affects gameplay, with storms creating hazardous conditions while calm seas provide opportunities for resource gathering. I implemented a ship upgrade system that allows players to customize their vessel with various weapons and defensive capabilities.

My Contributions

  • Developed enemy AI with behavior trees for varied combat encounters
  • Implemented procedural generation system for ocean environments and island placement
  • Created weapon systems with different firing patterns and upgrade paths
  • Designed and implemented the dynamic weather system affecting gameplay
  • Built ship customization and upgrade mechanics
  • Optimized rendering for large open-water environments
  • Implemented save/load system for player progression
  • Created visual effects for combat and environmental interactions

Greg On The Beach

Unreal Engine Blueprints Game Jam Rapid Prototyping

Project Overview

Greg On The Beach was developed during the Global Game Jam 2024 with the theme "Make Me Laugh". The game follows Greg, a character who just wants to enjoy his pizza at home but finds himself constantly harassed by pesky seagulls on the beach.

As part of a small team, I was responsible for character movement, seagull AI, and the core gameplay loop. The development process was intense, with the entire game created in just 48 hours. We focused on creating a humorous experience with simple but engaging mechanics.

The game features physics-based interactions where players must navigate Greg through various beach obstacles while protecting his pizza from the determined seagulls. The comedic elements were enhanced through exaggerated animations and sound effects that I helped implement.

My Contributions

  • Implemented character movement and physics-based interactions
  • Created seagull AI with different behavior patterns and attack strategies
  • Developed the core gameplay loop and progression system
  • Designed and built multiple beach environments with increasing complexity
  • Implemented comedic visual and audio elements to enhance the humorous theme
  • Created particle effects for sand, water, and seagull interactions
  • Optimized performance for the short development timeline
  • Integrated user interface elements for score and pizza health tracking
